# HP Trait Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add an `HP X` trait to player characters with a hearts UI, and mark characters dead when HP reaches zero.

**Architecture:** `HPTrait` is a factory (`createHpTrait(maxHp)`) that returns `{ id: 'hp', maxHp }`. The character model gains `hp` and `dead` fields. A new `simulation/hp.js` module checks for death each tick. The player panel gains a hearts row rendered from `char.hp` / `trait.maxHp`.

**Tech Stack:** Vanilla JS ES modules, Canvas API, DOM. No build step — verification is visual in the browser.

> **Note on testing:** This project has no automated test infrastructure. Every "verify" step means opening `index.html` in a browser (or refreshing) and checking visually.

### Task 4: Create `simulation/hp.js` — death check

**Files:**
- Create: `simulation/hp.js`

- [ ] **Step 1: Create the module**

```js
// simulation/hp.js
function hasHpTrait(character) {
  if (!character || !Array.isArray(character.traits)) {
    return false;
  }

  return character.traits.some((trait) => trait === 'hp' || trait?.id === 'hp');
}

export function updateHp(characters) {
  if (!Array.isArray(characters) || characters.length === 0) {
    return;
  }

  for (const character of characters) {
    if (!hasHpTrait(character)) {
      continue;
    }

    if (character.dead) {
      continue;
    }

    if (character.hp <= 0) {
      character.dead = true;
      character.frozen = true;
    }
  }
}
```

- [ ] **Step 3: Add helper functions for HP display**

Add the following helper functions before `createCard`:

```js
function getHpTrait(character) {
  return (character?.traits ?? []).find(
    (trait) => trait?.id === 'hp' || trait === 'hp',
  ) ?? null;
}

function buildHpRow(character) {
  const hpTrait = getHpTrait(character);
  if (!hpTrait) {
    return null;
  }

  const maxHp = typeof hpTrait === 'object' && typeof hpTrait.maxHp === 'number'
    ? hpTrait.maxHp
    : 3;
  const currentHp = typeof character?.hp === 'number' ? character.hp : maxHp;
  const filledCount = Math.max(0, Math.min(maxHp, Math.ceil(currentHp)));

  const row = document.createElement('div');
  row.className = 'player-card__hp';

  const label = document.createElement('span');
  label.className = 'player-card__hp-label';
  label.textContent = 'HP';

  const hearts = document.createElement('span');
  hearts.className = 'player-card__hp-hearts';
  hearts.setAttribute('aria-label', `${filledCount} of ${maxHp} HP`);

  for (let index = 0; index < maxHp; index += 1) {
    const heart = document.createElement('span');
    heart.className = index < filledCount
      ? 'hp-heart hp-heart--filled'
      : 'hp-heart hp-heart--empty';
    heart.textContent = index < filledCount ? '❤' : '♡';
    hearts.append(heart);
  }

  const counter = document.createElement('span');
  counter.className = 'player-card__hp-counter';
  counter.textContent = `${filledCount}/${maxHp}`;

  row.append(label, hearts, counter);
  return row;
}
```
